# 🚀 Moe Kyaw Aung - HTML/CSS Portfolio

A stunning, modern, fully responsive portfolio website built with pure HTML, CSS, and JavaScript. Features beautiful 3D animations, smooth scroll effects, and a living modern color palette.

## ✨ Features

### 🎨 Design & UI
- **Modern Color Palette**: Vibrant gradients with purple, pink, and cyan
- **3D & Particle Animations**: Smooth, eye-catching animations throughout
- **Responsive Design**: Perfect on mobile, tablet, and desktop
- **Dark/Light Mode**: Toggle between themes
- **Smooth Scrolling**: GSAP-quality animations with vanilla CSS/JS
- **Professional Layout**: Clean, modern, and user-friendly

### 📱 Sections

1. **Navigation Bar**
   - Sticky navigation with active state
   - Mobile-responsive hamburger menu
   - Theme toggle (dark/light mode)
   - Smooth scroll to sections

2. **Hero Section**
   - Animated profile image with orbit effect
   - Animated text reveal
   - Call-to-action buttons
   - Social media links
   - Scroll indicator

3. **About Section**
   - Professional bio
   - Key statistics
   - Timeline of achievements
   - Hover animations

4. **Skills Section**
   - 6 skill categories
   - Progress bars
   - Skill tags with hover effects
   - Grid layout

5. **Portfolio Section**
   - 6 featured projects
   - Filter by category
   - Project overlay on hover
   - Technology tags
   - Links to projects

6. **Testimonials Section**
   - Client testimonials
   - Star ratings
   - Profile pictures

7. **Contact Section**
   - Contact methods (email, phone, location)
   - Contact form
   - Social media links
   - Form validation

8. **Footer**
   - Quick links
   - Social media
   - Copyright information

### 🛠 Technologies

**Frontend:**
- HTML5 (semantic markup)
- CSS3 (advanced styling and animations)
- JavaScript ES6+ (interactivity)

**Tools & Libraries:**
- Font Awesome 6 (icons)
- Google Fonts (typography)
- No external animation libraries (pure CSS & JS)
